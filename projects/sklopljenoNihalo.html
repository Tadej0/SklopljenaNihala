Izbrana poglavja računalništva z didaktiko: Seminarska naloga

O projektu


V sklopu predmeta "Izbrana poglavja računalništva z didaktiko" sva izdelala interaktivno spletno simulacijo. Simulacija predstavlja navidezno demonstracijo nihanja dveh sklopljenih nihal. Zaradi težavnega razumevanja, pri obravnavanju omenjene teme, je dodatna ponazoritev primerna ideja za uporabo informacijsko komunikacijskih tehnologij.
Glavna prednost projekta pa se pokaže v interaktivnosti simulacije, ki omogoča uporabniku spreminjanje teže posameznih krogel ter kota, pod katerima se simulacija prične. Posledično je možna neomejena kombinacija raznolikih zanimivih poskusov, ki so dobro primerljivi s poskusi v resničnem svetu. Nadgradnja simulacje bi vključevala dodajanje nihal, spreminjanje dolžine posameznih nihal ter gravitacije.

Uporabljena tehnologija


Za realizacijo projekta sva uporabila več različnih spletnih tehnologij, jezikov in knjižnic. Osnovo spletne strani drži HTML5, ki pa je namenjen le za vključitev meta podatkov, JS in Bootstrap elementov ter CSS sloge. S pomočjo Bootstrap knjižnice sva razdelila spletno stran na dinamično mrežo(grid) ter dodala funkcijo modal, ki služi kot dodatno okno.
Znotraj canvas značke sva vgradila JS skripto, ki je grafično ponazori simulacijo, pri čemer upošteva zahtevane parametre. Zaradi zahtevnosti matematičnih izračunov sva znotraj JS skripte potrebovala dodatno knjižnico Math.
Kot je razvidno s statistične analize celotne spletne strani, vidne na strani Github, je ta sestavljena iz 55 % HTML, 32 % JavaScript in 13 % CSS kode.
